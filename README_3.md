## Missão

Usar inteligência artificial e engenharia de software para resolver problemas reais de pessoas reais — em Salvador e região.

Duas frentes, um propósito:

- **Projetos de impacto social** — aplicações que beneficiam comunidades de baixa renda, famílias em situação de vulnerabilidade e causas que o mercado ignora
- **Consultoria para pequenas empresas** — modernização de sistemas legados e introdução de IA acessível para PMEs que não têm time técnico interno

---

## Por que Salvador?

O Brasil tem milhões de pequenas empresas rodando sistemas dos anos 90, processos manuais e planilhas como banco de dados. Em Salvador, esse cenário é ainda mais acentuado — e a lacuna entre o que a tecnologia pode fazer e o que as PMEs locais têm acesso é enorme.

A SSAIT nasce para fechar essa lacuna. Com a mesma tecnologia usada por grandes bancos e fintechs, adaptada para a realidade e o orçamento de pequenos negócios.

## Frente 2 — Consultoria para PMEs

### O problema que resolvemos

Pequenas empresas em Salvador têm sistemas legados que travam o crescimento — ERPs desatualizados, processos manuais, planilhas Excel como banco de dados. Sabem que precisam modernizar, mas não têm time técnico nem orçamento para uma grande consultoria.

A SSAIT entra como parceiro técnico acessível — sem o custo de uma consultoria grande, com a qualidade de quem trabalha em sistemas financeiros críticos.

### Serviços

**Diagnóstico de sistema legado**
Mapeamento do sistema atual, identificação de gargalos e proposta de modernização por etapas — sem reescrever tudo de uma vez.

**Introdução de IA no negócio**
Automação de processos repetitivos, atendimento inteligente, análise de dados e geração de relatórios — usando IA acessível e adaptada para a realidade da PME.

**Desenvolvimento de MVPs**
Validação rápida de ideias com stack moderna — do problema ao produto funcional em semanas, não meses.

**Integração de sistemas**
APIs, integrações entre sistemas existentes, migração de dados — sem parar a operação.

### Stack de consultoria

```
Backend:   Java 21 · Spring Boot · Quarkus · Python
Frontend:  Angular · React
IA / LLMs: Anthropic API · OpenAI · LangChain
Dados:     PostgreSQL · MySQL · SQL Server · Oracle
DevOps:    Docker · CI/CD · Azure DevOps
Cloud:     AWS · GCP
Legado:    PHP · Laravel · Delphi · sistemas ERP
```
